Cookie Policy

Last updated: 2026-03-12. This page lists storage currently used by the website implementation.

Storage in use

Key Type Purpose Duration
rosa-theme localStorage Stores user selected dark/light mode. Until manually cleared
rosa-language localStorage Stores language preference (EN/ES/DE). Until manually cleared
rosa-cookie-consent localStorage Stores cookie consent choice. Until manually cleared

No analytics cookies by default

This implementation does not set non-essential analytics or marketing cookies by default. If analytics tools are added later, this page and the consent logic must be updated before release.

How users can manage storage

Users can clear browser cookies/local storage via browser settings at any time. You should additionally provide a visible "cookie settings" entry if non-essential trackers are introduced.
